# khudebarta-sms-api

Khudebarta SMS API is a simple and lightweight PHP-based API designed for integrating Khudebarta bulk SMS services into applications.  
This API allows developers to send SMS efficiently using secure and easy-to-configure endpoints.

## Features
- Bulk SMS sending support
- Simple PHP integration
- Lightweight and fast
- Easy configuration
- Suitable for websites and web applications

## Requirements
- PHP 7.4 or higher
- cURL enabled
- Active Khudebarta SMS account
- Internet connection
